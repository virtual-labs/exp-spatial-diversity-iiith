Instructions

To understand the different combining techniques in SIMO, follow these steps:


  1. Enter the value for number of receive antennas(1-10)
  2. Enter the value of Noise Variance (N0)
  3. Select the combining technique you want to use
  4. Click on the calculate button to get the desired outputs in the Observations.
  5. Please refresh the page to restart the experiment
