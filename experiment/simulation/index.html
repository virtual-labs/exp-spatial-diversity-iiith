Single Input Multiple Output

Instructions

To understand the different combining techniques in SIMO, follow these steps:


  1. Enter the value for number of receive antennas (Nr)
  2. Enter the value of Noise Variance (N0)
  3. To obtain the channel realisation, click on "Get Channel"
  4. Select the combining technique you want to use
  5. Enter the value for SNR Threshold
  6. Click on the calcularte button to get the desired outputs in the Observations.
